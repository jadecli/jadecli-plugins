name: doctor
description: Diagnose issues with installed dev tools and system settings
---

Run the devtools doctor to check for configuration problems and resource issues.

## Instructions

1. Execute the doctor script:

   ```bash
   bash devtools-plugin/scripts/doctor.sh
   ```

2. Present findings organized by severity (FAIL > WARN > PASS).
3. For each issue, suggest the specific fix command.
