# DevTools Research

Research index for macOS developer tooling optimized for Claude Code + monetizable web app development.

## Organization

### By Device Surface
- `surfaces/macos-system.md` -- Kernel, launchd, file descriptors, energy settings
- `surfaces/menubar.md` -- Menubar-resident apps (Amphetamine, Maccy, Flow, etc.)
- `surfaces/dock-desktop.md` -- Dock behavior, Spaces, window management
- `surfaces/terminal.md` -- Shell config, SSH, CLI tools
- `surfaces/network.md` -- Proxyman, VPN, API testing
- `surfaces/storage-db.md` -- Database GUIs, Docker volumes, disk management

### By Claude Code Entry Point
- `entry-points/cli.md` -- Direct CLI usage, shell integration, env vars
- `entry-points/mcp.md` -- MCP servers and browser automation
- `entry-points/hooks.md` -- Pre/post tool use hooks
- `entry-points/skills.md` -- Skills that leverage installed tooling
