# Claude Code MCP Entry Point

Tools that integrate through Model Context Protocol servers.

## Browser Automation (claude-in-chrome)

MCP server for Chrome browser control. Benefits from:
- Proxyman: inspect traffic generated by MCP browser actions
- Figma: design reference open in one tab while MCP automates another

## Docker MCP

Docker provides an official MCP server:
```json
{
  "mcpServers": {
    "docker": {
      "command": "docker",
      "args": ["mcp", "server"]
    }
  }
}
```

Allows Claude Code to manage containers, images, volumes through MCP protocol
instead of shell commands.
