# Menubar Surface

Apps that live in the macOS menubar for persistent background functionality.

## Amphetamine (App Store ID: 937984704)
- Prevents system/display sleep
- Key settings: Start Session At Launch, Allow Closed-Display Sleep = off
- Default duration: indefinite
- Claude Code relevance: prevents sleep during long autonomous coding sessions

## Maccy (App Store ID: 1527619437)
- Clipboard history with fuzzy search
- Keyboard-first, no mouse required
- Ignores password manager entries
- Claude Code relevance: constant copy-paste of error output, file paths, code snippets

## Flow (App Store ID: 1423210932)
- Pomodoro/focus timer
- Blocks distracting apps and websites during focus sessions
- Exports session logs to CSV
- Claude Code relevance: time-boxes long sessions, reduces context-switching

## SnippetsLab (App Store ID: 1006087419)
- Code snippet manager with menubar quick-access
- 600+ language syntax highlighting
- GitHub Gist sync
- Claude Code relevance: store reusable patterns (auth boilerplate, Stripe handlers, Docker fragments)

## 1Password (App Store ID: 1333542190)
- Password manager with menubar integration
- SSH agent built-in (replaces ssh-agent)
- CLI integration via `op` command
- Claude Code relevance: secrets management, SSH key management, `.env` value retrieval via CLI
